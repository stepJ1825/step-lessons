wait() - освобождает монитор и переводит поток в состояние ожидания

notify() - пробуждает один случайный поток из ожидающих

notifyAll() - пробуждает все ожидающие потоки

join() - ожидает завершения указанного потока

yield() - предлагает планировщику переключиться на другой поток

interrupt() - устанавливает флаг прерывания потока

Все эти методы используются для координации работы между потоками и управления их выполнением.
